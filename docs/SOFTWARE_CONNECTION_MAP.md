# SOFTWARE CONNECTION MAP

ChatGPT -> produit la décision ou le découpage de lot  
AXEND/state -> stocke l'état minimal utile  
Cursor -> exécute la prochaine action dans le repo  
Scripts AXEND -> génèrent handoff / restart / proof  
Ollama -> peut produire des résumés ou handoffs locaux  
Git/GitHub -> capture la réalité du code via commit et historique

## Règle
Aucune connexion magique.  
Tout passe par :
- le repo
- les JSON d'état
- les handoffs
- les preuves
